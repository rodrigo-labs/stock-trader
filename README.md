Stock Trader
==============
Implementação do projeto de finalização do curso vuejs 2 da udemy.

### Tecnologias usadas
1. Vue2
2. VueRouter
3. VueEx
4. axios
5. google firebase

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

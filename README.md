Stock Trader
==============
Projeto final do curso vuejs 2 da udemy.
Um simulador de compra e venda de ações.

### Tecnologias usadas
1. Vue2
2. VueRouter
3. VueEx
4. Vuetify
5. axios
6. google firebase

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
